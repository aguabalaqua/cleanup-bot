# Contributing

Thank you for your interest in contributing to CLEANER.

## Development Setup

Clone the repository:

```bash
git clone https://github.com/aguabalaqua/cleanup-bot
cd cleanup-bot
npm install
```

Run the agent:

```bash
npm run start
```

## Guidelines

- Follow the existing TypeScript project structure
- Keep modules small and composable
- Document new agent modules in the `/docs` directory
- Maintain consistency with the agent architecture

## Reporting Issues

If you encounter a bug or unexpected behavior, please open a GitHub issue including:

- description of the issue
- expected behavior
- relevant logs if available
- steps to reproduce the problem
